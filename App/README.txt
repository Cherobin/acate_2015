###################
#   Instalação    #
###################
1) Esse projeto necessita da instalação dos seguintes softwares:
    - Apache 2
    - MySQL 5.5
    - PHP 5
    - PHP 5 - GD
    - PHP 5 - MySQL Driver

2) Necessário os arquivos para a pasta web do apache
3) Setar permissão dos de leitura aos arquivos, e permissão de leitura e escrita na pasta 'upload' e 'upload/qrcode'
4) Habilitar o mod_rewrite do apache e configurar para funcionar na pasta web
5) Faça a restauração do arquivo 'database.sql' para o MySQL
6) Modifique o arquivo index.php e altere as configurações da aplicação
